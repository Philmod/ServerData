Welcome to ServerData !

 

This is a web server that listens to datas in JSON format, and stores it as time-series in Apache Cassandra.
The visualization is done through this simple web page.
Node.js, Apache Cassandra, Express.js, Socket.io, Backbone.js, Twitter Bootstrap, Highcharts.
